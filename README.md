**TaskZen Frontend**
TaskZen is a freelancer platform. This repository contains the frontend of the TaskZen project, built with JSF (JavaServer Faces), HTML, Java, JavaScript, and Tailwind CSS.

**Features**

User Registration & Login: Users can sign up as clients or companies and log in using the frontend.
Interactive Design: Responsive and modern UI/UX designed using Tailwind CSS with JSF for server-side rendering.
REST API Integration: Communication with the backend using JavaScript Fetch API and JSF for data exchange.
Form Validation: Frontend form validation implemented using JavaScript for enhanced user experience.
Animations: Smooth and responsive animations using CSS and JavaScript for interactive feedback.
Task Posting & Bidding: Clients can post tasks, and companies or freelancers can bid on available tasks.
Responsive Design: Optimized for both desktop and mobile viewports.
